Resumen CSS

CSS es el lenguaje usado para darle formato a las paginas HTML, cosa que anteriormente no era posible solo con HTML ya que este tenia como propsito unicamente la modificacion de contenido.

La sintaxis principal de CSS es h1 (indicador de elemento) {Propiedad:valor;pripiedad:valor;etc.}. La importancia de los atributos van en cascada y hay 3 maneras de darle estilo a los elementos, aqui listadas en orden de importancia.

  1. Estilo inline - Se coloca dentro de la etiqueta inicial del elemento.
  2. Hojas de estilo internas y externas - Se colocan en < head>
  3. Predeterminadas del navegador

Que los estilos vayan en cascada significa que el navegador le va a dar prioridad al ultimo estilo aplicado a los mismos atributos

Selector id
Es usado para darle un determinado formato e identificacion a un elemento dentro de la pagina. Para declarar estos id se colocan en Style con #Nombredeid { atributos} y se llaman colocando id="Nombredeid" dentro de la etiqueta inicial del elemento.

Clases
Las clases se usan para aplicar u mismo formato a diferentes elementos en la pagina sin la necesidad de declararlo en el apartado Style o de darle el mismo estilo a todos los elementos del mismo tipo en toda la pagina (como pasa en el apartado Style). Igual que el id se declara en Style con .NombredeClase {atributos} o colocando p.center {} para declarar que solo los elementos < p> con clase "center" van a adquirir los atributos declarados; se llaman a un elemento te la misma manera que el id pero colocando class.
Recuerda que puedes agrupar los elementos a los que quieres aplicar un mismo fomato separandolos por comas antes de los atributos; los comentarios en CSS se colocan entre /* y */

Los colores se especifican de tres maneras: un nombre de color valido de la lista, con una formula RGB(0,0,0) o con el codigo hexadecimal del color, todo esto se puede encontrar aqui.
