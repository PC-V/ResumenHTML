Resumen CSS

CSS es el lenguaje usado para darle formato a las paginas HTML, cosa que anteriormente no era posible solo con HTML ya que este tenia como propsito unicamente la modificacion de contenido.

La sintaxis principal de CSS es h1 (indicador de elemento) {Propiedad:valor;pripiedad:valor;etc.}. La importancia de los atributos van en cascada y hay 3 maneras de darle estilo a los elementos, aqui listadas en orden de importancia.

  1. Estilo inline - Se coloca dentro de la etiqueta inicial del elemento.
  2. Hojas de estilo internas y externas - Se colocan en < head>
  3. Predeterminadas del navegador

Que los estilos vayan en cascada significa que el navegador le va a dar prioridad al ultimo estilo aplicado a los mismos atributos

Selector id
Es usado para darle un determinado formato e identificacion a un elemento dentro de la pagina. Para declarar estos id se colocan en Style con #Nombredeid { atributos} y se llaman colocando id="Nombredeid" dentro de la etiqueta inicial del elemento.

Clases
Las clases se usan para aplicar u mismo formato a diferentes elementos en la pagina sin la necesidad de declararlo en el apartado Style o de darle el mismo estilo a todos los elementos del mismo tipo en toda la pagina (como pasa en el apartado Style). Igual que el id se declara en Style con .NombredeClase {atributos} o colocando p.center {} para declarar que solo los elementos < p> con clase "center" van a adquirir los atributos declarados; se llaman a un elemento te la misma manera que el id pero colocando class.
Recuerda que puedes agrupar los elementos a los que quieres aplicar un mismo fomato separandolos por comas antes de los atributos; los comentarios en CSS se colocan entre /* y */

Los colores se especifican de tres maneras: un nombre de color valido de la lista, con una formula RGB(0,0,0) o con el codigo hexadecimal del color, todo esto se puede encontrar aqui.

Fondos
El fondo se define con background-propiedad: valor, estas propiedades pueden ser color, si quieres definir un color como fondo; imagen, si quieres definir una imagen como fondo; repeat, para definir el numero de repeticiones de la imagen en la pagina; position, para definir la posicion de la imagen en la pagina y attachment para definir si la imagen se quedará fija en la pagina mientras se navega o si se mueve.

Bordes Los bordes se colocan con border-style: valor; los bordes pueden ser puntos (dotted), lineas (dashed), solido (solid), doble (double), sombreado (groove o ridge), inset y outset. El color se determina con border-color y se le puede aplicar un estilo diferente a cada borde con border-top/bottom/left/right-style; para ahorrarte codigo puedes poner las cuatro propiedades de cada borde despues del border-style, las propiedades se aplicaran en el siguiente orden:

  • 4 propiedades - Arriba, derecha, abajo, izquierda.
  • 3 propiedades - Arriba, izquierda y derecha, abajo.
  • 2 propiedades - Arriba y abajo, izquierda y derecha.
  • 1 propiedades - Todos

Al igual que con los bordes, los margenes pueden tener propiedades en sus 4 lados pero estos son usados para generar espacio dentro de un elemento y puede tener los siguientes valores: auto, para que el navegador cacule el margen automaticamente; lenght, determina el tamaño del margen y colapse, que hace 0 el espacio entre el margen y el borde; puedes tambien aplicar el mismo margen que el elemento en el que esta anidado otro con el valor inherit. El padding funciona igual que los margener pero se refiere al espacio generado alrededor de un objeto. El outline es un 'borde' fuera del borde y puede tener las mismas propiedades que este.

Texto
Se puede modificar el estilo del texto de diversas maneras; se le puede aplicar un color, alineación (Text-align), decoraciones (text-decoration), transformarlo a todo mayusculas o minusculas (text-transformation), agregarle sangria (text-indent), espaciado entre cada letra (letter-spacing), la altura de las lineas (line-height), la direccion del text (direction), espacio entre palabras (word spacing), sombreado (text-shadow). El tipo de letra se modifica con font-atributo; estos atributos pueden ser family(tipo de letra) style (normal, italicas, oblicua), size (tamaño), weight (grosor)

Puedes modificar el estilo de los links dentro de < style> especificandolos con a:link {} el formato del link normalmente, con a:visited {} el formato del link una vez que fue abierto y a:active{} el formato del link cuando esta activo, con estos formatos puedes construir botones de lonk para darle mayor estetica a tu página

Para darle cierto formato a las listas se hace de la misma manera, dandole diferentes clases a cada lista para especificar el estilo de cada una; se usa list-style-atributo para personalizar los marcadores de cada lista, se puede usar type (tipo de marcador), image (coloca una imagen como marcador) y position (acomoda la lista en una posicion especifica)

Tablas
Para modificar el estilo de las tablas se refiere proncipalmente a los bordes; puedes usar collapse para reducir a 0 el espacio entre el margen y el borde, especificar el borde que se desea modificar, usar la propiedad hover para que cambie de color cuando se coloca el puntero sobre la tabla, usar el indicador nth-child(even/odd) para darle un formato a rayas o usar overflow-x:auto para permitirle a la tabla ser responsiva.

Puedes darle un limite a los objetos aplicando un max-width para evitar que estos se esparzan por todo el ancho de la pagina y definir la posicion de un elemento con position

Overflow
La propiedad overflow se usa cuando un elemnto no cabe dentro del bloque y te permite navegar (scroll) por el, verlo (visible), ocultarlo (hidden) o colocar el scroll solo si es necesario (auto).

Para colocar bloques alineados los unos con los otros se puede usar la propiedad float para darle un estilo de "cajas", usando despues la propiedad clear para especificar que pasa despues de esto; otra forma es simplemente usar la propiedad inline-block dentro de float.

Alineacion
Para alinear los elementos a la izquierda, centro de un bloque simplemente se pone margin: auto; text-align:center colocaria el texto exactamente en el centro del bloque y si quieres alinear al centro una imagen simplemente la conviertes en un bloque con display: block y usas margin:auto. Otra manera de alinear el contenido de un bloque puede ser position:absolute (siempre definiendo margen y padding), con float para alinearno a la izquierda o derecha, con padding superior e inferior para ajustar la altura o con line-height.

Selectores y combinadores
Los selectores de CSS pueden contener selectores simples y combinadores, como por ejemplo:

  • Descendientes - se coloca solo un espacio entre el bloque y el elemento para especificar que los atributos dentro de {} se van a aplicar a todos los elementos p anidados dentro del padre, en este caso.
  • Hijos - se coloca un > entre el bloque y elemento para especificar que los atributos entre {} se van a aplicar unicamente a los elementos que descienden directamente del padre.
  • Hermanos adyacentes - se coloca un + entre el bloque y elemento para especificar que los atributos entre {} se van a aplicar unicamente a los elementos que siguen del hijo aunque no desciendan directamente del padre.
  • Hermanos generales - se coloca un + entre el bloque y elemento para especificar que los atributos entre {} se van a aplicar unicamente a los elementos anidados dentro del padre en el mismo nivel.

Las pseudo-clases y pseudo-elementos se aplican especificamente a elementos y se ponen con : despues del elemento, la lista de pseudoclases puedes encontrarla aqui.

Una barra de navegacion se puede implementar colocando una lista con links en vez de texto, estas se pueden colocar verticalmente, dentro de un bloque u horizontal escribiendo inlini en el display de la lista.

Desplegables
Un desplegable se coloca haciendo una clase para el objetivo, una para el desplegable y haciendo que el deplegable se ejecute al poner el mouse sobre el objetivo, aqui se muestra un ejemplo para entender la estructura: 

 < style>
/* Style The Dropdown Button */
.dropbtn {
    background-color: #4CAF50;
    color: white;
    padding: 16px;
    font-size: 16px;
    border: none;
    cursor: pointer;
}

/* The container 
 - needed to position the dropdown content */
.dropdown {
    position: relative;
    display: inline-block;
}

/* Dropdown Content (Hidden by Default) */
.dropdown-content {
    display: none;
    position: absolute;
    background-color: #f9f9f9;
    min-width: 160px;
    box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
    z-index: 1;
}

/* Links inside the dropdown */
.dropdown-content a {
    color: black;
    padding: 12px 16px;
    text-decoration: none;
    display: block;
}

/* Change color of dropdown links on hover */
.dropdown-content a:hover {background-color: #f1f1f1}

/* Show the dropdown menu on hover */
.dropdown:hover .dropdown-content {
    display: block;
}

/* Change the background color of the dropdown button when the dropdown content is shown */
.dropdown:hover .dropbtn {
    background-color: #3e8e41;
}

Contadores
Los contadores son variables fijas dentro de la pagina que pueden incrementarse con reglas de CSS, las propiedades que se usan son:

  • Counter-reset - crea o reinicia un contador
  • Counter-incremeent - Incrementa el valor de un contador
  • Content - Inserta contenido generado
  • Counter - Agrega el valor del contador a un elemento

Un ejemplo de uso de contador puede ser el siguiente: 

 < style>
body {
    counter-reset: section; //  Crea el contador llamado Section. 
}

h2::before { // define que la regla del contador como el uso del encabezado h2.
    counter-increment: section;  //Se va a incrementar el contador Section cada que se cumpla la regla. 
    content: "Section " counter(section) ": "; // Va a instertar antes de el encabezadoel texto "Section #: ", siendo "#" el contador.
}
< /style>
