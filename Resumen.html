Resumen HTML

Todos los documentos HTML deben empezar con la declaracion del tipo, que en este caso es < !DOCTYPE html> ademas de empezar con < html> y terminar con < /html>.

La parte visible del documento se coloca entre < body> y < /body>; para insertar un texto se pueden utilizar encabezados (que inician con < h1> y terminan con < /h1>), que varian de tamanio siendo h1 el mas grande, h6 el mas pequenio y hr un cambio tematico que se ve como una linea, y parrafos, en los que simplemente se coloca el texto entre < p> y < /p>.

Para insertar un link se pone el destino despues del atributo "href" entre < a> y < /a> de la siguiente manera:
< a href="destino"> Nombre del link < /a>

Las imagenes se definen con la etiqueta < img> y se insertan de la siguiente manera: < img src="Nombre de la imagen en sistema" alt="Titulo" width="ancho en pixeles" height="largo en pixeles">

Atributos

El atributo de idioma se declara dentro de la etiqueta < html> y se declara como lang="Codigo del idioma", por ejemplo < html lang="en-US">
El titulo mostrara el texto introducio al colocar el puntero sobre el parrafo y se coloca dentro de la etiquetta < p> asi: < p title="Texto del titulo">
El atributo href, como se habia visto anteriormente, se usa para insertar links dentro de un texto.
Los atributos de tamaño se colocan dentro de la etiqueta de imagen y son width y height.

Independientemente del numero de lineas que ocupe un texto en el codigo, estas no se veran a menos que se introduzcan en parrafos separados, se use el elemtento < pre> o se use un separador de elemento vacio o break, que es: < br>.

Estilos

El estilo se define para cada etiqueta y puede contener los siguientes elementos que se escrben con la sintaxis < etiqueta style="propiedad:valor;">:

El color de fondo se establece con "Background-color:nombre de color;".
El olor de texto se establece con "Color:nombre de color;".
El tipo de letra se define con "font-family:tipo de letra;".
El tamaño de la letra se define con "font-size: porcentaje de aumento;".
La alineacion se define con "text-align:posicion;".

Formato de texto

Los elementos de formato de texto se coloca dentro del parrafo abriendo y cerrando normalmente la etiqueta, los siguientes son los tipos de formato que se pueden aplicar 

 

    < b> - Bold text
    < strong> - Important text
    < i> - Italic text
    < em> - Emphasized text
    < mark> - Marked text
    < small> - Small text
    < del> - Deleted text
    < ins> - Inserted text
    < sub> - Subscript text
    < sup> - Superscript text

Citas

Una cita normal se coloca entre < q> y < /q> dentro del parrafo.
UNa cita de bloque se usa para citas largas y se coloca con la etiqueta < blockquote cite:"Referencia de la cita"> y, como siempre, cerrandola con < /blockquote>.
Las abreviaciones se usan para que, cualdo se coloque el puntero sobre esta, aparezca el significado de la abreviacion; un ejemplo seria < p>The < abbr title="World Health Organization">WHO< /abbr> was founded in 1948.
La eticqueta < cite> se usa para refererire al nombre de la obra y se coloca dentro del parrafo

Nota: Si quieres colocar un comentario dentro del codigo debes ponerlo asi: < !-- Aqui tu comentario -->.
Puedes modificar el estilo de los links dentro de < style> especificandolos con a:link {} el formato del link normalmente, con a:visited {} el formato del link una vez que fue abierto y a:active{} el formato del link cuando esta activo; tambien puedes poner el link dentro de una imagen colocandola dentro de < a> y < /a> en vez del texto del link. Recuerda que siempre que el link lleve a una pagina externa debes de escribir al principio http://

Tablas

Para formar una tabla se coloca la eqiqueta < table>, con < tr> se define las filas de la tabla < th> los encabezados y < td> el contenido de cada celda.
Para agregarle un estilo se coloca en el apartado de style al principio del código y se coloca dentro de las llaves despues de table, td, th {} para toda la tabla, despues de th para los encabezados y despues de td para los datos de celda.
Los estilos que se le pueden dar a cada tabla son los siguientes:

Atributo Característica
Border Tamaño del borde, forma y color
Border-collapse Colapsa los bordes de la tabla y las celdas.
Padding Determina el espacio entre el borde y el texto
Text-align Determina la posicion del texto dentro de la celda>
border-spacing Determina el espacio entre los bordes.
colspan="#" Se usa para indicar que una celda contiene un numero de columnas.
rowspan="#" Se usa para indicar que una celda contiene un numero de filas.
caption Determina el titulo de la tabla.
