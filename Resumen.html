Resumen HTML

Todos los documentos HTML deben empezar con la declaracion del tipo, que en este caso es < !DOCTYPE html> ademas de empezar con < html> y terminar con < /html>.

La parte visible del documento se coloca entre < body> y < /body>; para insertar un texto se pueden utilizar encabezados (que inician con < h1> y terminan con < /h1>), que varian de tamanio siendo h1 el mas grande y h6 el mas pequenio, y parrafos, en los que simplemente se coloca el texto entre < p> y < /p>.

Para insertar un link se pone el destino despues del atributo "href" entre < a> y < /a> de la siguiente manera:
< a href="destino"> Nombre del link < /a>

Las imagenes se definen con la etiqueta < img> y se insertan de la siguiente manera: < img src="Nombre de la imagen en sistema" alt="Titulo" width="ancho" height="largo">

Atributos

El atributo de idioma se declara dentro de la etiqueta < html> y se declara como lang="Codigo del idioma", por ejemplo < html lang="en-US">
El titulo mostrara el tetxo introducio al colocar el puntero sobre el parrafo y se coloca dentro de la etiquetta < p> asi: < p title="Texto del titulo">
El atributo href, como se habia visto anteriormente, se usa para insertar links dentro de un texto.
Los atributos de tamanio se colocan dentro de la etiqueta de imagen y son width y height.
